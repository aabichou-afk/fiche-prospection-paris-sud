Les 3 réflexes

100% financé

« C'est votre budget formation OPCO, déjà cotisé. 0 € pour vous. »

B2B transport only

Jamais de particuliers. Jamais le mot « CPF ». Jamais de garantie d'emploi.

Une fiche / appel

Ce qui n'est pas saisi n'existe pas. Remplissez la fiche pendant l'appel.

Rappel conformité : on ne se présente jamais comme « OPCO Mobilité ». On dit : « Paris Sud Formation, organisme habilité OPCO Mobilité ». Le réseau/communauté : on n'en parle qu'à l'adhésion.
